Membranous quantification (cytoplasmic-ring, for CD8 / TIM-3)
Measures DAB in the cytoplasmic ring using each marker's calibrated completeness cutoffs (Calibrate tab) instead of nuclear DAB — the correct compartment for membranous markers. Needs a calibration per marker; without one it falls back to the weaker ring-mean rule.

Cross-type analysis measures whether CD8+ and TIM-3+ cell populations are spatially associated across registered serial sections, using the calibrated intensity-reweighted inhomogeneous cross-K — which tests for association BEYOND a shared tissue preference (both cell types favouring the same architecture), the failure mode of a naive model (see ihc.md §15). This is a population-level statistic and does not imply single-cell co-expression, which requires multiplex imaging on the same section.

Fit membrane cutoffs to your own slides
DAB staining isn't quantitative — "how much brown = positive" differs by antibody, protocol and scanner, so cutoffs from another dataset won't transfer. Segment your stained slides, click positive and negative cells, and fit the cutoffs for this marker. Do this once per staining protocol, then switch between saved calibrations in Quant.
Add one or more representative slides — pooling several fields captures the staining variability a single image misses, and the callability score is reported as an honest leave-one-cell-out estimate.
